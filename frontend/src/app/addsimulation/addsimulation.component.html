General Initialization

Edit Simulation

The initialization part of the Water Tariff Dashboard involves configuring several essential components to define the socio-economic simulation environment.

How to proceed: First the primitives, then the population, then usage tarification, then the demand and finally, the launch.

  1. Tariff Module Initialization (Primitives) - Configure the water pricing scheme to be evaluated
  2. Population Module Initialization - Load the household-level data needed for microsimulation
  3. Usage Tarification - Configure water usage pricing for Eau Potable and Assainissement
  4. Demand Module - Compute expected consumption
  5. Launch - Run the simulation
