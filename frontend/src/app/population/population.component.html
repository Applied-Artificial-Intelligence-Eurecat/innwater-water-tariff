Generate Household Population Sample

At this stage, you must generate a household population sample, which will be created according to the Réunion 2020 data file.

Choose Source Database :

Select a value between 1,000 and 10,000 in increments of 1,000.
Select a value between 0.01 and 0.20.

The standard deviation is a value that will be used to modify the income of the randomly selected household.
